The Craft

I build systems for coordination and execution.

I've been working on Caret^, an open source semantic notation for agent workflows. It's not a framework trying to solve chaos by adding mass. It solves it by adding shape.

The work is real engineering: a working spec, formal semantics, a security model, and governance. On external testing with GStack, it delivers a 95.8% token reduction benchmark—that matters when you're coordinating multiple agents at scale.

I also built and run EA-OS, a custom agent-based executive assistant I use in my own work every day. It's proof that these systems work when they're built with discipline.

This section should feel like opening someone's workshop. You see what they're making. Not selling. Building.

The Passion
