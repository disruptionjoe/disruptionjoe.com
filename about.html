What I'm Building

I find patterns that produce structure.

Caret^ is an open-source semantic notation for agent workflows in Markdown. The core problem it solves is expressivity. When you speak, you have volume, timbre, and tone. You signal the intensity and shape of an idea without thinking about it. In text, especially in prompts, that dimension is mostly missing.

Caret^ brings it back. Scalar directives let you dial intent the way you would adjust your voice: ^depth8 ^skeptical7 says "go deep and push back" the way a raised eyebrow and a slower cadence would in a room. Four levels of caret scope let you adjust context, change the lens, spawn a separate worker, or start fresh.

It's practical because it is written in Markdown files, so you don't have to learn a programming language. And it might save you 50% of the tokens you would have used.

Most prompt systems try to solve chaos by adding mass. Caret^ solves it by adding shape.

I am also building CapacityOS, a modular agent-based operating system I use daily to run my own work. It is my fourth-generation personal agent harness, which I plan to open-source soon. It includes:

  1. A cadence-driven orchestration system that fires the right work at the right rhythm, from hourly to weekly, without manual context-switching.
  2. A structured work lifecycle with explicit human-agent handoff control, so agents stay bounded and nothing drifts.
  3. A dashboard-first steering layer where five surfaces keep attention on what matters and maintain full decision history.

I've been personalizing agent systems for fun since leading an agent marketplace standards project in Q1 2025.

The Bigger Picture
